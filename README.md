# Portf-lio-de-Murilo
O objetivo é diferir todo o conhecimento que foi obtido através das práticas de atividades das aulas de Programação Web II.

<h1> Conteúdos estudados </h1>
<h3> Uso do GitHub </h3>
<li> Aprendemos a utilizar o GitHub para criar repositórios de códigos e para manter o site online </li>
<li> Criação do Portfólio para a obtenção da nota bimestral </li>

<h3> Criação de sites com Google App Script </h3>
<li> Aprendemos a utilizar a ferramenta "Google App Script" para a criação de sites, cuja ferramenta permitia o uso de diversas funções distintas do google de modo em que todas as informações sejam armazenadas em um arwuivo chamado "código.gs" </li>

<h3> Criação de Tabelas </h3>
<li> Nesta aula aprendemos a criar tabelas, cujas mesmas visam organizar informações obtidas anteriormente em forma de tabela </li>
<li> As tabelas podem ser criadas através da tag "table" e suas tags de separação, sendo elas "td", usada para a criação de colunas, e "tr", usada para a criação de elementos da coluna </li>
